Perancangan dan Pengembangan Aplikasi Uncle Do Barbershop Berbasis Web untuk Digitalisasi dan Aksesibilitas Layanan Jasa

Penulis: Al Qudri, Asrofiyatul Aini, Irpan Sinaga, Yudy Purnama

Afiliasi: Program Studi Ilmu Komputer, BINUS Online Learning, Universitas Bina Nusantara

1. Pendahuluan

Transformasi digital layanan barbershop diperlukan untuk mengurangi antrean, meningkatkan pemesanan, dan menghadirkan pengalaman pelanggan modern. Penelitian ini merancang aplikasi web Uncle Do Barbershop yang memadukan reservasi daring, ulasan pelanggan, serta dashboard admin.

2. Arsitektur Sistem

Frontend: Laravel Blade + Tailwind CSS Backend: Laravel (MVC) Database: MySQL

Arsitektur memanfaatkan pemisahan lapisan presentasi dan logika bisnis, memudahkan skalabilitas serta pemeliharaan. Tailwind CSS memastikan konsistensi UI, sementara MySQL mengelola data reservasi dan ulasan.

3. Use Case Utama

  • Online reservation (order) dengan pemilihan jadwal dan barber.
  • Customer review untuk layanan dan pengalaman potong rambut.
  • Admin dashboard untuk manajemen layanan, jadwal, dan analitik.

4. Metodologi Penelitian

Pendekatan Custom Scrum dengan iterasi pendek memastikan verifikasi fungsional setiap inkrement.

Sprint Planning
Development & Testing

5. Implementasi Fitur

  • Order / Booking: formulir jadwal, pilihan barber, notifikasi pemesanan.
  • Customer Review: rating layanan, komentar, validasi pengguna.
  • Dashboard Admin: monitoring order, manajemen jadwal, dan laporan cepat.
